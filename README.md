## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/glzjoo/Iloilo-Farmers-Hub.git
cd Iloilo-Farmers-Hub
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
