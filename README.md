# 🌾 Iloilo Farmers Hub

A digital marketplace connecting local farmers in Iloilo directly with consumers — promoting fair pricing, direct communication, and sustainable agricultural trade.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** — fast dev server & build tool
- **Tailwind CSS 3** — utility-first styling
- **React Router v7** — client-side routing
- **Firebase** — backend services

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [Git](https://git-scm.com/)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/glzjoo/Iloilo-Farmers-Hub.git
cd Iloilo-Farmers-Hub
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```


## Available Scripts

| Command           | Description                        |
| ----------------- | ---------------------------------- |
| `npm run dev`     | Start local development server     |
| `npm run build`   | Build for production               |
| `npm run preview` | Preview the production build       |
| `npm run lint`    | Run ESLint to check for issues     |
